# Customer Segmentation Analysis

## Project Overview
This project performs Customer Segmentation Analysis using Python.  
The program groups customers based on their spending behavior and visit frequency.

It helps businesses identify:
- High-value customers
- Medium customers
- Low-engagement customers
The project also visualizes customer data using charts.

Pandas → used to create and manage dataset.
Matplotlib → used for graphs.
Customer Segmentation:
High → Spending ≥ 7000
Medium → Spending between 3000 and 6999
Low → Spending < 3000
High-value customers → customers spending more money.
Low-engagement users → customers visiting less frequently.
Histogram → shows spending distribution.
Pie Chart → shows customer category percentage.
